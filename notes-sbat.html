This week I open-sourced some Rust code for working with UEFI SBAT: https://github.com/google/sbat-rs

There are two crates in this project:

  • sbat - A no-std library for parsing SBAT and doing revocation checks.
  • sbat-tool - A command-line utility for working with SBAT.
